# 🎮 Tic-Tac-Toe (Juego del Gato)

Un juego de "El Gato" moderno y responsivo construido con **Vite** y **JavaScript (Vanilla)**. El proyecto aplica una arquitectura de software limpia y sigue principios de diseño UX/UI para ofrecer una experiencia de usuario fluida.

## 🚀 Características
- **Interfaz Responsiva:** Adaptable a dispositivos móviles y escritorio.
- **Arquitectura MVC:** Separación clara entre Modelos, Vistas y Controladores.
- **Voz y Sonido:** Integración de servicios de audio para una experiencia interactiva.
- **Temas:** Soporte para cambio de temas (claro/oscuro).
- **Rápido:** Configurado con Vite para un tiempo de carga y desarrollo casi instantáneo.

## 🛠️ Tecnologías utilizadas
* **Frontend:** HTML5, CSS3 (Variables, Flexbox, Grid).
* **Lógica:** JavaScript (ES6+).
* **Herramienta de Construcción:** [Vite.js](https://vitejs.dev/)
* **Control de Versiones:** Git & GitHub.

```text
src/
